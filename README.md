<h1># The Sync Point - Blog Application</h1>

A Django-based blog platform where users can create, edit, and manage blog posts.

<h3>## Features</h3>
- User authentication
- Create, edit, delete blog posts
- Multiple categories
- Draft and published states
- Profile management
- Image upload
- Search functionality

<h3>## Installation</h3>
1. Clone the repository
2. Create virtual environment
3. Install requirements
4. Run migrations
5. Start server

<h3>## Technologies</h3>
- Django 6.0.1
- Python 3.13
- SQLite
- HTML/CSS/JavaScript
